支持的Markdown语法

点击查看飞书文档中的Markdown语法总结

基本语法

  • 标题 (#, ##, ###)
  • 加粗 (**文字**)
  • 斜体 (*文字*)
  • 删除线 (~~文字~~)
  • 列表 (- 或 1.)
  • 任务列表 (- [x] 或 - [ ])
  • 图片 (![描述](URL))
  • 表格
  • 引用 (>)
  • 代码块 (```)
  • 内联代码 (`代码`)
  • 链接 ([]())
  • 脚注 ([^1])
  • 水平分割线 (***)

高级功能

  • Mermaid图表 (```mermaid)
  • PlantUML图表 (```plantuml)
  • 代码高亮 (支持多种语言)
  • 卡片式布局
  • 拖拽排序
  • 缩放和拖拽图表
  • 响应式设计
  • HTML反向转Markdown

Mermaid示例

```mermaid
graph TD
    A[开始] --> B{条件判断}
    B -->|是| C[执行操作1]
    B -->|否| D[执行操作2]
    C --> E[结束]
    D --> E
```

PlantUML示例

```plantuml
@startuml
Alice -> Bob: 你好
Bob --> Alice: 嗨，最近怎么样？
Alice -> Bob: 我很好，谢谢！
@enduml
```
