# Markdown to HTML Converter with PlantUML Support

![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-yellow)
![PlantUML](https://img.shields.io/badge/PlantUML-Supported-brightgreen)
![Mermaid](https://img.shields.io/badge/Mermaid-Supported-9cf)

A powerful, all‑in‑one Markdown editor and document converter that runs entirely in your browser.  
It provides a live preview, supports **PlantUML** and **Mermaid** diagrams, mathematical formulas (MathJax), task lists, tables, footnotes, and much more. You can export your document as **editable HTML**, **read‑only HTML**, **PDF**, or **Markdown**.

![Screenshot](docs/screenshot.png) *(Add a screenshot of the tool here)*

## Features

- 📝 **Live Markdown editing** with two‑column layout (input + preview)
- 🎨 **6 document themes** – default, tech, minimal, professional, nature, dark
- 📊 **Diagram support** – PlantUML (via plantuml.com) and Mermaid (local rendering)
- 🧮 **Mathematical formulas** – inline `$...$` and block `$$...$$` via MathJax
- 🖱️ **Drag & drop** – reorder document nodes (headings and their content)
- ✏️ **In‑place editing** – edit title and content directly in the preview
- 🔍 **Diagram controls** – zoom, pan, fullscreen for Mermaid & PlantUML
- 💾 **Auto‑save** – content is saved to localStorage every 30 seconds
- 📄 **Export options**
  - Markdown (.md)
  - HTML – editable (preserves all interactive features)
  - HTML – read‑only (clean, ideal for sharing)
  - PDF (with diagram rendering)
- 🔁 **Reverse conversion** – import HTML (generated by this tool) back to Markdown
- ⌨️ **Keyboard shortcuts** – bold, italic, code, headings, lists, preview, export
- 📑 **Table of contents** – auto‑generated, collapsible, with edit/delete entries
- 🖼️ **Media placeholder** – click to select local image/audio/video files (converted to base64)

## How to Use

1. **Open the HTML file** in any modern web browser (Chrome, Firefox, Edge, Safari).
2. Write or paste your Markdown into the left panel.
3. Click **"Generate Preview"** (or press `Ctrl+G`) to see the rendered document.
4. Use the toolbar buttons to **import/export**, **change themes**, or **clear** the editor.
5. Drag document nodes to reorder them. Click the edit (✏️) icon to modify titles/content directly.
6. Export your final document as Markdown, HTML, or PDF.



### Supported Markdown Syntax

- Headings (`#`, `##`, `###`, …)
- Bold (`**text**`, `__text__`), Italic (`*text*`, `_text_`), Strikethrough (`~~text~~`)
- Inline code (`` `code` ``), code blocks (```` ```language ````) with syntax highlighting
- Unordered lists (`- item`) and ordered lists (`1. item`)
- Task lists (`- [ ] todo`, `- [x] done`)
- Tables (using `|` and `|-` for alignment)
- Blockquotes (`> text`)
- Images (`![alt](url)`), audio/video via file picker (click placeholder)
- Links (`[text](url)`)
- Footnotes (`[^1]`)
- Horizontal rules (`***` or `---`)
- Math formulas (`$E=mc^2$`, `$$\int x^2 dx$$`)

### Diagrams

**Mermaid** – wrap your diagram code in ```` ```mermaid ````  
**PlantUML** – wrap your diagram code in ```` ```plantuml ````  
(PlantUML diagrams are rendered via the official plantuml.com server; an internet connection is required.)

## Dependencies (CDN)

All libraries are loaded via CDN – no local installation required:

- [TailwindCSS](https://tailwindcss.com/)
- [Font Awesome 6](https://fontawesome.com/)
- [Mermaid 10.6.1](https://mermaid.js.org/)
- [PlantUML Encoder 1.4.0](https://github.com/markushedvall/plantuml-encoder)
- [html2pdf.js 0.10.1](https://github.com/eKoopmans/html2pdf.js)
- [MathJax 3](https://www.mathjax.org/)

## Limitations

- **PlantUML** requires an active internet connection (calls plantuml.com).
- **PDF export** uses `html2pdf.js`; complex diagrams or extremely long documents may render slowly.
- **Reverse conversion** (HTML → Markdown) works only for HTML files generated by this tool.
- **Local media files** (images, audio, video) are converted to base64 and embedded – very large files may cause performance issues.
